## 概述 

外观模式（Facade），也叫门面模式，提供了一个统一的接口，用来访问子系统中的一群接口。

它定义了一个高层接口，让子系统更容易使用。
外观模式是迪米特法则非常好的典型的例子。
降低了应用层（client），与子系统之间的耦合度。
