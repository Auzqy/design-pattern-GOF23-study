
## 概述 

模版方法模式（Templete Method），定义了一个算法的骨架，并容许子类为一个或多个步骤提供实现。

模版方法使得子类可以在不改变算法结构的情况下，重新定义算法的某些步骤。

模版方法的扩展——钩子方法
钩子方法提供了缺省的行为，子类可以在必要时进行扩展。
